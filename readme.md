## Plugins para WordPress con React y la WP REST API

Cada ejemplo va guiando desde la preparación del entorno de desarrollo JavaScript con Webpack y Babel, hasta obtener los datos a través de la WP REST API con React y mostrarlos.

### Herramientas

[Yarn](https://yarnpkg.com)

[Webpack](https://webpack.js.org)

[Babel](https://babeljs.io)

[React](https://facebook.github.io/react)

[WordPress](https://wordpress.org)
